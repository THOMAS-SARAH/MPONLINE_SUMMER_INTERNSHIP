The Language used to write the codes are C#(C Sharp)

C# (pronounced "C-sharp") is a modern, general-purpose, and object-oriented programming language developed by Microsoft.
It runs on the .NET framework and is one of the most popular languages for building a wide range of applications. 


# Key Features:-
*Object-Oriented*: It follows core OOP principles like encapsulation, inheritance, and polymorphism.
*Type-Safe*: C# prevents errors by ensuring that variables are used according to their specific data types.
*Cross-Platform*: With modern .NET, you can build apps that run on Windows, macOS, and Linux.
*Modern Syntax*: Its syntax is similar to other C-family languages like Java and C++, making it familiar to many developers.
